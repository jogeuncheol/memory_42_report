### ft_substr
- **형식** `substr(char const *s, unsigned int start, size_t len)`
- **반환 값** `char *`형 데이터
- **설명**\
문자열 `s`에서 `start`의 인덱스부터 의 문자열을 `len` 만큼 반환한다.\
동적 할당 실패시 `NULL`을 반환한다.
- **주요 코드**
```c
ptr = (char *)malloc(sizeof(char) * (len + 1));
if (!ptr)
    return (NULL);
while (i < len)
{
    *(ptr + i) = *(s + start + i);
    i++;
}
*(ptr + i) = '\0';
return (ptr);
```
